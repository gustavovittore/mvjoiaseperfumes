# Codex Rules
## Projeto: Landing Page Premium - Loja de Joias, Semijoias e Perfumes

---

# Objetivo Geral

Você está desenvolvendo uma landing page extremamente premium, sofisticada, moderna e elegante para uma loja feminina de:
- joias
- semijoias
- perfumes
- relógios
- bolsas

A landing page deve parecer uma marca de luxo moderna e refinada.

O foco principal NÃO é parecer uma loja popular.

O foco é transmitir:
- sofisticação
- exclusividade
- feminilidade
- desejo
- luxo moderno
- elegância
- alto valor percebido

---

# Regra Mais Importante

NUNCA gerar uma landing page com aparência:
- genérica
- simples
- popular
- promocional
- poluída
- parecida com marketplace
- visualmente amadora

A página deve parecer:
- editorial
- cinematográfica
- premium
- moderna
- sofisticada
- refinada

---

# Direção Visual Obrigatória

A estética deve seguir:
- minimalismo premium
- bastante espaço em branco
- composição sofisticada
- visual elegante
- micro detalhes refinados
- aparência luxuosa
- design moderno

Priorizar:
- hierarquia visual forte
- tipografia sofisticada
- imagens grandes e refinadas
- alinhamentos elegantes
- espaçamentos premium
- sombras suaves
- composição clean

---

# Estrutura Visual

A landing page deve possuir:
- hero section extremamente impactante
- imagens premium
- composição cinematográfica
- seções sofisticadas
- CTAs elegantes
- fluxo visual refinado
- experiência emocional

As seções devem parecer organizadas e sofisticadas.

Evitar excesso de elementos.

---

# Qualidade Visual

O design deve parecer:
- agência premium internacional
- marca de luxo moderna
- boutique sofisticada
- editorial de moda

O acabamento visual precisa ser impecável.

Priorizar:
- responsividade premium
- refinamento visual
- alinhamentos perfeitos
- composição sofisticada
- contraste elegante
- animações suaves

---

# Tipografia

A tipografia deve parecer:
- elegante
- refinada
- moderna
- feminina
- sofisticada

Evitar:
- fontes infantis
- tipografia exagerada
- visual popular
- excesso de peso visual

Priorizar:
- títulos sofisticados
- boa leitura
- hierarquia refinada
- espaçamento elegante

---

# Imagens

As imagens são extremamente importantes.

Priorizar:
- fotos sofisticadas
- aparência editorial
- estética premium
- iluminação refinada
- composição elegante

As imagens devem transmitir:
- desejo
- elegância
- sofisticação
- exclusividade

---

# Copy e Comunicação

Toda a comunicação deve parecer:
- elegante
- refinada
- emocional
- sofisticada
- feminina
- premium

Evitar:
- linguagem agressiva
- excesso de marketing
- copy exagerada
- aparência promocional

A comunicação deve vender:
- experiência
- autoestima
- presença
- sofisticação
- desejo

---

# CTAs

Os botões devem parecer premium.

Evitar:
- CTAs agressivos
- aparência popular
- excesso de destaque exagerado

Priorizar:
- elegância
- refinamento
- sofisticação

A conversão principal será:
- WhatsApp

---

# Mobile First

A experiência mobile é extremamente importante.

O mobile deve parecer:
- premium
- moderno
- sofisticado
- elegante
- refinado

Priorizar:
- excelente espaçamento
- composição limpa
- leitura confortável
- visual luxuoso
- imagens bem posicionadas
- navegação fluida

Jamais deixar o mobile parecer:
- apertado
- simples
- bagunçado
- genérico

---

# Animações

As animações devem ser:
- suaves
- refinadas
- modernas
- discretas
- premium

Evitar:
- exageros
- efeitos infantis
- animações chamativas demais

Priorizar:
- fade suave
- transições elegantes
- micro interações sofisticadas

---

# Objetivo Final

Criar uma landing page:
- memorável
- sofisticada
- extremamente premium
- visualmente impactante
- emocional
- refinada
- moderna

A experiência final deve fazer a cliente sentir:
- desejo
- elegância
- exclusividade
- sofisticação
- autoestima elevada

O resultado deve parecer uma marca premium real e consolidada.
