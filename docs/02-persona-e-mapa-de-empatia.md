# Persona e Mapa de Empatia
## Projeto: Landing Page Loja Premium de Joias, Semijoias e Perfumes

---

# Objetivo do Projeto

Desenvolver uma landing page extremamente premium, sofisticada e moderna para uma loja de joias, semijoias, bolsas, relógios e perfumes femininos.

A página deve transmitir:
- luxo moderno
- sofisticação
- exclusividade
- elegância feminina
- desejo
- alto valor percebido

O foco NÃO é parecer uma loja popular.

A marca deve transmitir:
- experiência premium
- estética refinada
- elegância
- atendimento sofisticado
- presença visual forte
