# Page Structure
## Projeto: Landing Page Premium - MV Jóias & Perfumes

---

# Objetivo da Estrutura da Página

A landing page deve criar uma experiência:
- sofisticada
- elegante
- emocional
- feminina
- premium
- refinada
- editorial
- cinematográfica

A navegação deve conduzir a cliente por uma experiência de:
- desejo
- sofisticação
- autoestima
- exclusividade
- elegância moderna
- presença
- refinamento

A conversão principal da página será:
- contato via WhatsApp

---

# Novo Direcionamento Estrutural

A landing page NÃO deve seguir estrutura de:
- e-commerce tradicional
- catálogo comum
- marketplace
- grid excessivo de produtos

O objetivo agora é criar:
- uma experiência editorial premium
- uma boutique digital sofisticada
- uma apresentação visual refinada
- uma experiência de marca

A estrutura deve priorizar:
- poucas imagens por seção
- uma imagem principal premium por categoria
- bastante espaço visual
- composição limpa
- alinhamentos refinados
- storytelling visual
- copy emocional

A percepção premium deve vir:
- da curadoria
- da composição
- da elegância visual
- da sofisticação da experiência

Não da quantidade de produtos.

---

A landing page deve alternar entre:
- seções em azul petróleo ultra escuro
- seções brancas/off-white minimalistas

Paleta principal:
- Azul Petróleo Ultra Escuro: `#010F1B`
- Dourado Sofisticado: `#D1AF66`

A alternância visual deve criar:
- respiro visual

- experiência editorial premium

- hero azul petróleo ultra escuro com imagem premium ao fundo
- seção clara institucional

- bloco azul institucional
- seção clara de produto
- seção clara de produto
- seção clara de produto
