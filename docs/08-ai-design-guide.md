# AI Design Guide
## Projeto: Landing Page Premium - MV Jóias & Perfumes

---

# Objetivo do Arquivo

Este arquivo define toda a direção visual, estética e estrutural da landing page da MV Jóias & Perfumes.

Ele deve ser seguido obrigatoriamente para garantir que toda a página tenha:
- aparência premium
- sofisticação visual
- elegância feminina
- luxo moderno
- consistência de marca
- alto valor percebido
- experiência refinada
- excelente experiência mobile e desktop

A landing page deve parecer:
- uma boutique premium
- uma marca sofisticada
- uma experiência editorial feminina
- uma marca consolidada e elegante

Jamais deve parecer:
- template genérico
- marketplace
- catálogo comum
- loja popular
- e-commerce simples

---

MV Jóias & Perfumes é uma marca feminina premium especializada em:
