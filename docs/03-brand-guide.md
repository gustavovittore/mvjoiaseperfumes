# Brand Guide
## Projeto: MV Jóias & Perfumes

---

# Essência da Marca

A MV Jóias & Perfumes é uma marca feminina premium que une:
- sofisticação
- feminilidade
- autoestima
- elegância
- exclusividade
- luxo moderno

A marca busca proporcionar uma experiência refinada para mulheres que valorizam:
- presença
- autenticidade
- delicadeza
- beleza
- sofisticação

Mais do que vender acessórios, a MV deseja transmitir:
- confiança
- autoestima
- elegância
- identidade
- feminilidade

---

# Posicionamento da Marca

A MV Jóias & Perfumes NÃO deve parecer:
- popular
- promocional
- genérica
- visualmente poluída
- exageradamente comercial

A marca deve transmitir:
- exclusividade
- refinamento
- elegância feminina
- sofisticação moderna
- luxo silencioso
- experiência premium

A percepção da marca deve lembrar:
- boutiques premium
- joalherias sofisticadas
- marcas femininas de luxo
- estética editorial moderna

---

# Paleta de Cores Oficial

## Cor Principal

Azul sofisticado:
```css
#010F1B
```

Sensações transmitidas:
- profundidade
- sofisticação
- elegância
- luxo moderno
- autoridade
- refinamento

Essa cor deve ser predominante:
- hero section
- footer
- seções escuras
- fundos principais
- composição premium

---

## Cor Secundária / Destaques

Dourado sofisticado:
```css
#D1AF66
```

Utilizar em:
- botões
- detalhes premium
- ícones
- elementos sofisticados
- micro detalhes
- destaques elegantes

Sensações transmitidas:
- luxo
- exclusividade
- refinamento
- feminilidade premium

---

## Cor dos Textos Claros

Branco:
```css
#FFFFFF
```

Utilizar:
- textos sobre fundo azul
- títulos da hero
- contraste elegante
- áreas premium escuras

# Estratégia Visual da Landing Page

A landing page deve alternar entre:
- seções escuras sofisticadas
- seções claras minimalistas

Estrutura visual:
- hero azul sofisticado
- seção branca clean
- seção azul premium
- seção branca elegante
- footer azul sofisticado

Essa alternância deve criar:
- ritmo visual
- sofisticação
- respiro visual
- sensação editorial premium

---

# Tipografia

A tipografia deve parecer:
- elegante
- refinada
- sofisticada
- feminina
- moderna

Priorizar:
- excelente hierarquia visual
- bastante respiro
- títulos sofisticados
- leitura confortável
- composição premium

Evitar:
- fontes exageradas
- aparência popular
- visual infantil
